## 🔧 透明化処理スクリプト(コピペで使える)

```python
from PIL import Image
import numpy as np
import os

mapping = [
    ('xxx.png', '/output/weather_01_sunny.png',         '晴れ'),
    ('yyy.png', '/output/weather_02_partly_cloudy.png', '晴れ時々曇り'),
    # ... 7枚分
]

def get_bg_color(arr):
    corners = [arr[0:5, 0:5], arr[0:5, -5:], arr[-5:, 0:5], arr[-5:, -5:]]
    pixels = np.concatenate([c.reshape(-1, c.shape[-1]) for c in corners])
    return np.mean(pixels[:, :3], axis=0).astype(int)

def transparentize(img):
    img = img.convert('RGBA')
    arr = np.array(img)
    bg = get_bg_color(arr)
    
    tolerance = 35
    rgb = arr[:, :, :3].astype(int)
    diff = np.sqrt(np.sum((rgb - bg) ** 2, axis=2))
    
    arr[diff < tolerance, 3] = 0
    soft = (diff >= tolerance) & (diff < tolerance * 2)
    soft_alpha = np.clip(((diff - tolerance) / tolerance) * 255, 0, 255).astype(np.uint8)
    arr[soft, 3] = soft_alpha[soft]
    
    return Image.fromarray(arr, 'RGBA'), bg

for src, dst, label in mapping:
    img = Image.open(src)
    result, bg = transparentize(img)
    result.thumbnail((256, 256), Image.LANCZOS)
    result.save(dst, 'PNG', optimize=True)
```

**重要**:
- ChatGPT画像は背景がRGB(完全な白)のことが多い → tolerance=35で確実に透明化
- リサイズは256×256まで(UIで小さく表示するため)

## ⚠️ よくあるトラブル

### 透明化されない問題
- ChatGPT画像はRGB(透明レイヤーなし)の場合が多い
- 解決: `img.convert('RGBA')` を必ず最初に実行
- 確認: 四隅のα値が0、中央の絵がある部分のα値が200以上か

### 絵柄が薄く見える
- 元の水彩タッチが透明部分と境界で消えがち
- 解決: tolerance=35がベスト(40以上だと絵まで消える)
