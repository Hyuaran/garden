### 1. 税務コンプライアンス最優先
マイナンバー pgcrypto 暗号化、admin+ 復号 + 監査記録、法定期限（1/31）管理 7 段階リマインダー。税法上の責任に耐える設計。

### 2. 3 者対応 UI（従業員・事務・admin）
従業員ウィザード 2 段階（モバイル対応必須）、事務一覧 + 個別確認、admin 進捗ダッシュボード。Leaf C-03 パターンを年末調整向けに拡張。

### 3. 案 D 準拠の徹底
源泉徴収票 / 支払調書 / リマインダー、全て Chatwork 本文に URL 不流通、Garden ログイン誘導のみ。マイナンバー含む PDF は個人 Vendor にメール添付不可。

### 4. Bud 🔴 最厳格テスト
Tree 🔴 と同等 + PDF Visual Regression + XML XSD 検証 = 業界最厳格級。計算ロジック 100% カバレッジ必達。税理士事務所との結果突合 3 日連続 OK で本番 GO。

### 5. 段階展開の税理士連携
α（東海林さん）→ β（経理 2-3 人 + **税理士事務所立会い**）→ 本番。税理士の事前確認なしに本番投入しない設計。

### 6. 法改正への柔軟対応
税率・控除額をマスタテーブル駆動（`bud_tax_deduction_master` 等）、admin UI で編集可能。毎年の税制改正に SQL 変更のみで対応。

### 7. e-Tax XML はスコープ調整
Phase C-1 は CSV 出力（税理士事務所経由）、e-Tax XML 直接提出は Phase C-2 または Phase D。XSD 入手と検証が前提。
