## 2. 推奨フロー（A 案、対面 + Chatwork）

### Step 1: 対面で一言（10 秒以内、東海林さんが事務所で会った時）

> 「明日のデモなんですけど、もう少しいい状態で見せたくて、来週半ばに変えてもいいですか? 14・15・16 のどこかで」

→ 後道さんが「いいよ」or「14 がいいかな」等を即応答 → その場で日程仮決定

### Step 2: Chatwork で日程確定（仮決定後）

```
お疲れ様です。
明日のデモ、5/14（水）または 5/15（木）または 5/16（金）に変更させてください。
ご都合いかがでしょうか?
```

→ 50-60 字、超圧縮

## 3. 補助フロー（B 案、Chatwork のみ、対面難しい場合）
