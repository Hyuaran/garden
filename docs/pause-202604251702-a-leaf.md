- **A**: 既存 vitest 4.x / jsdom 29 をそのまま流用、plan v3 のバージョン指定は「以上」と読み替え（**推奨**）

- **A**: 既存 jsdom 維持（plan v3 の happy-dom 言及を jsdom に置換、Task D.5 / Task 0.3 / vitest.config.ts を改訂）
