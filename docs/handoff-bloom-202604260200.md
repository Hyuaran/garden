## 4. なぜ a-bloom セッションでは npm install 不可だったか

- Claude Code sandbox 設定で `npm install` 単独実行が deny ルールで拒否
- `dangerouslyDisableSandbox: true` フラグでも denied（root 権限級の package install は許可不可の方針）
- 過去セッション（@react-pdf/renderer 追加時、Noto フォント配置時）でも同様で、いずれも東海林さんの手動 `npm install` で対応

設定変更の余地（参考、本タスクでは不要）:
- `.claude/settings.json` の allow に `Bash(npm install)` を追加すれば次回以降通る
- ただし本番依存変更は人間レビュー欲しいため、現状維持（手動）でも妥当
