## 4. セクション 3: 中段 3 カード（法人数 / 従業員数 / 総資産集計）

mock 通常版では中段に 3 つ並ぶ（横一列）:

| カード | 値 | アイコン |
|---|---|---|
| **法人数** | 6 社 | 緑系アネモネアイコン |
| **従業員数** | 248 名 | 緑系人型アイコン |
| **総資産集計** | ¥428,600,000 | （金額アイコン）|
