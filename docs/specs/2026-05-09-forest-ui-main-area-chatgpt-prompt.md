### 1-2. Bud / Bloom precedent（世界観統一の参考）

- 左サイドバー: デュアル（12 モジュール nav-apps + モジュール内 nav-pages）
- 右サイドバー: 状態ウィジェット（Bloom = ShojiStatusWidget）
- タブ実装: URL ベース（`/forest/[tab]`）
- カラー: Forest 固有 = #1b4332 / #2d6a4f / #40916c（緑系）+ 6 法人アネモネカラー
- 装飾: 角丸 14-20px、シャドウ、バックドロップ blur(8px)、進行期 glow アニメ

### 1-3. 制約

- ヘッダー / 左右サイドバー = **既決**（Bud / Bloom 流用、本依頼スコープ外）
- 既存 Forest UI（v9.html セクション）= なるべくそのまま、世界観だけ合わせる
- メイン部分（左右サイドバー中央）のみ本依頼の対象
- 追加項目・タブ = ガンガン提案歓迎
