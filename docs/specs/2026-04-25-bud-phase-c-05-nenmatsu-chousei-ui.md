## 1. 目的とスコープ

### 目的

従業員（本人）・事務担当者・admin の 3 者がそれぞれの役割で利用する年末調整 UI を提供。**2 段階ウィザード + 進捗ダッシュボード**で、紙ベースの年末調整を完全電子化。

### 含める

- 従業員向け: 年末調整入力ウィザード（2 段階、Leaf C-03 パターン）
- 事務担当者向け: 従業員一覧 → 個別確認 → 差戻/確定 フロー
- admin 向け: 全従業員進捗ダッシュボード
- 証明書添付 UI（Leaf 関電画像 upload パターン踏襲）
- 源泉徴収票発行画面（C-02 連携）
- Garden マイページでの PDF 表示

### 含めない

- データモデル詳細（C-01）
- PDF 生成ロジック（C-02）
- 支払調書 UI（C-03 側）
- 法定調書合計表 UI（C-04 側）
- テスト（C-06）

---

## 2. UI フロー概要

### 2.1 3 者の役割

```
従業員（本人）
  ↓ 入力（11 月〜12 月）
  年末調整申告書ウィザード → 添付 → 提出
  ↓
事務担当者（staff）
  ↓ 確認（12 月）
  内容確認 → 差戻 or 承認
  ↓
事務 / manager / admin
  ↓ 確定（12 月末）
  給与計算反映 → 源泉徴収票発行
  ↓
従業員（本人）
  ↓ 受領（1 月）
  マイページで源泉徴収票 DL
```

### 2.2 画面一覧

| 対象 | 画面パス | 機能 |
|---|---|---|
| 従業員 | `/bud/my-page/nenmatsu-chousei` | ウィザード入力・添付・提出 |
| 従業員 | `/bud/my-page/gensen-choshu` | 源泉徴収票表示・DL |
| 事務 | `/bud/backoffice/nenmatsu` | 従業員一覧 + 検索 + 一括操作 |

## 7. モバイル対応

### 7.1 対応範囲

- 従業員入力ウィザード: **スマホ対応必須**（年末調整は自宅・通勤中に記入されるため）
- 事務担当者画面: タブレット / デスクトップ対応（スマホ対応は優先度低）
- admin ダッシュボード: デスクトップ / タブレット

### 7.2 スマホ UX

- ウィザードは 1 ステップ = 1 画面（スクロール必要だが複雑 UI 排除）
- 証明書アップロード: カメラ直接撮影（heic 対応）
- Step 間は自動保存（通信断での喪失防止）

## 9. アクセシビリティ

### 9.1 フォーム

- 金額入力: `type="text" inputMode="numeric"` + 自動 カンマ区切り
- ラベルと input の明確な紐付け（`htmlFor` / `id`）
- エラー表示 `role="alert"`（known-pitfalls §3.2 遵守）
- Tab 順序: 上から下、Step 1 → Step 2

### 9.2 スクリーンリーダー

- ステップ進捗: `aria-current="step"`
- 必須欄: `aria-required="true"`
- 添付ファイル: `aria-label="証明書ファイル 3 件添付中"`

## 11. テスト観点（詳細 C-06）

- ウィザード自動保存・ブラウザ閉じ復元
- 必須欄バリデーション境界
- 添付 5MB 超過 / heic → jpg 変換
- 差戻フローと再提出の状態遷移
- 一括承認の失敗ハンドリング
- モバイル縦画面 / 横画面 / タブレット
- RLS: 他人のウィザードに直接 URL 入力しても見えない

## 12. 判断保留事項

- **判1: 添付ファイル数上限**
  - 1 カテゴリ 3 枚 / 5 枚 / 無制限
  - **推定スタンス**: カテゴリごと 5 枚、全体 20 枚まで
- **判2: 自動保存の頻度**
  - 5 秒毎 / 入力完了時 / 手動のみ
  - **推定スタンス**: 入力完了時（blur イベント）+ 手動「保存して中断」
- **判3: 計算結果のリアルタイム表示**
  - 表示する / 最終計算は事務確定時
  - **推定スタンス**: 概算を表示、「確定額は事務確認後」と注記
- **判4: 差戻時の編集可能範囲**
  - 全欄再入力 / 差戻指摘箇所のみ
  - **推定スタンス**: 全欄再入力可（但し前回値は保持表示）
- **判5: スマホ撮影時の heic**
  - heic 直接保存 / 自動 jpg 変換
  - **推定スタンス**: 自動 jpg 変換（Canvas + heic2any、UI-03 踏襲）
- **判6: 事務の一括承認の権限**
  - staff でも可 / manager 以上
  - **推定スタンス**: manager 以上（ミスリスク高いため、§project_configurable_permission_policies で変更可）
- **判7: 期限後の提出可否**
  - 強制ロック / admin のみ追加可
  - **推定スタンス**: admin unlock で追加可（遅刻者対応）
- **判8: 英語対応**
  - 日本語のみ / 英語併記
  - **推定スタンス**: 日本語のみ（外国人従業員は admin が個別対応）
