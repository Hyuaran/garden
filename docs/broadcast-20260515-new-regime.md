| 状況 | 再起動タイミング |

1. 現在の作業を安全な状態にする (commit + push、または handoff md 起草)

4. 再起動後、最初に **`.codex/rules.md`** と **`docs/known-pitfalls.md`** を Read してから作業に戻る
