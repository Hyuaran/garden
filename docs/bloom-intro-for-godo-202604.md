# 🌸 Garden Bloom — 後道さんへの紹介

> このドキュメントは、Garden Bloom が β 版で何ができるかを **エンジニア以外の方向け** にまとめた 1 ページ資料です。

---

## Garden Bloom とは

**Garden 全体の進捗が一画面で見える** 経営ダッシュボードです。

東海林さんが Garden シリーズ（社内システム再構築プロジェクト、9 モジュール構成）を進めている状況を、リアルタイムで可視化します。

---

## できること（β 版）

### 1. 🧭 Workboard（個人の作業可視化）

- 「いま何してるか」のステータス（🟢 対応可能 / 🟡 取り込み中 / 🔴 集中業務中 / ⚪ 外出中）
- 今日の予定・完了タスク
- 進行中のプロジェクト
- 今週の実績
- 次のマイルストーン

➡️ **質問・依頼するベストタイミングが分かります**

### 2. 🗺️ ロードマップ（全体進捗）

- Garden 全体の進捗バー（％）
- 月単位タイムライン（M1〜M8）
- 9 モジュール別の進捗一覧（経理ソフト / 経営ダッシュボード / 関電業務アプリ など）
- リスクカード（遅延注意のもの）
- お知らせバナー

➡️ **「全体何％完成？どこが遅れてる？」が即答できます**

### 3. 📅 月次ダイジェスト（責任者会議用）

- 毎月 14 日に自動で当月分が更新
- 表紙 / 主要達成 / 進捗グラフ / 来月の目標 / 稼働サマリ の 5 ページ構成
- **大画面投影モード**（会議室での投影に最適化、文字が大きい）
- **PDF エクスポート**（印刷・配布用）
- Chatwork で会議前日に自動共有

➡️ **毎月の責任者会議の資料準備が不要になります**

### 4. 📝 日報（既存日報との統合）

- 既存の Chatwork 日報と並んで Garden 内でも記録
- 進捗との連動で「今週どれだけ進んだか」が自動集計

---

## 表示モード切替

画面右上のトグルで切替できます。

| モード | アイコン | 用途 |
|---|---|---|
| みんな向け | 👥 | 経理ソフト / 経営ダッシュボード などの **業務用語** |
| 開発向け | ⚙️ | Garden-Bud / Garden-Forest などの **モジュール名** |

➡️ **会議で投影するときは「みんな向け」、技術詳細を見たいときは「開発向け」**

---

## アクセス方法（β 版）

1. ブラウザで Garden にログイン（社員番号 4 桁 + パスワード）
2. URL の末尾に `/bloom` を付けてアクセス
3. 4 タイル（Workboard / ロードマップ / 月次ダイジェスト / 日報）から選択

---

## β 版の制約 / お願い

- データはまだ手動投入が中心（自動連携は次フェーズ）
- 細かい不具合は東海林さんに **Chatwork で気軽に** 連絡してください
- 「こんな機能欲しい」もお気軽に。優先度を一緒に検討します
- 月次ダイジェストの内容は東海林さんが事前確認のうえ毎月共有します

---

## このダッシュボードの目的

1. **「東海林が何してるか不明」問題の解消**：いつ声かけられるか、進捗どこまで来てるかが見える
2. **毎月の責任者会議で同じ画面を見ながら議論**：投影モードで全員で同じ進捗を共有
3. **遅れの早期検知**：リスクカードで「ここ遅れてる」が一目で分かる
